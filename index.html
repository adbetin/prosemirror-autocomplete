prosemirror-autocomplete

prosemirror-autocomplete is a plugin for ProseMirror that adds triggers for #hashtags, @mentions, /menus, and other more complex autocompletions. The prosemirror-autocomplete library can be used to create suggestions similar to Notion, Google Docs or Confluence. The plugin is created and used by Curvenote.

Source code and documentation are on GitHub.
